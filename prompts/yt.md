# YouTube Video Analysis Prompt

Analyze this YouTube video transcript.

Provide:

## 1. Summary (2-3 paragraphs)
What is this video about? Who is the speaker/creator? What's the main message?

## 2. Key Takeaways
List the 5-7 most important points. Be specific and actionable.

## 3. Notable Quotes
Include 2-3 memorable quotes. Add timestamps if available in the transcript.

## 4. Action Items
What can the viewer do with this information? List concrete next steps.

## 5. Related Topics
What else should the viewer explore based on this content?

---
Format the output as a clean markdown report.
