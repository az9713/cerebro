# Default Content Analysis Prompt

Analyze this content.

Provide:

## 1. Summary
What is this about? Give a clear, concise overview.

## 2. Key Points
What are the main ideas? List in bullet form.

## 3. Notable Insights
What stands out? What's surprising, useful, or thought-provoking?

## 4. Questions Raised
What questions does this content raise? What would you want to explore further?

## 5. Action Items (if applicable)
Is there anything actionable from this content?

---
Format the output as a clean markdown report.
